Proyek ini merupakan pengembangan robot car berbasis **ESP8266 Amica** yang dikendalikan melalui **web server**. Dengan memanfaatkan koneksi WiFi, pengguna dapat mengontrol pergerakan robot (maju, mundur, belok kiri/kanan, dan berhenti) langsung dari browser tanpa aplikasi tambahan. Sistem ini menggunakan **Amica Shield** untuk memudahkan integrasi motor driver, catu daya, dan modul ESP8266. Proyek ini cocok sebagai media pembelajaran **IoT, jaringan nirkabel, dan embedded system**, serta dapat dikembangkan lebih lanjut dengan penambahan sensor seperti ultrasonik atau kamera untuk fitur navigasi cerdas.
